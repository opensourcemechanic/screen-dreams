# Mobile Guide

Write anywhere with Screen Dreams' mobile-optimized interface.

## Mobile Compatibility

Screen Dreams is designed to work great on mobile devices with responsive design and touch-friendly controls.

### Supported Browsers

✅ **Chrome Mobile (Android)** - Recommended  
✅ **Safari Mobile (iOS)**  
✅ **Firefox Mobile**  
✅ **Edge Mobile**

## Recommended Setup

### Horizontal Orientation Required

<div class="alert alert-warning">
  <i class="fas fa-mobile-alt me-2"></i>
  <strong>Important:</strong> For the best experience, use your phone in horizontal (landscape) orientation.
</div>

The application is designed primarily for desktop and horizontal mobile use. While some features work in vertical orientation, the editor and screenplay management interfaces are optimized for horizontal viewing.

### What Works Best in Horizontal Mode

- ✅ **Editor**: Full screenplay text with syntax guide visible
- ✅ **Screenplay Cards**: Proper button layout and spacing
- ✅ **Scene Content**: Optimal scrolling and readability
- ✅ **AI Panel**: Full functionality with prompt editing

### Vertical Mode Limitations

- ⚠️ **Editor**: Syntax guide moves below text area (reduced functionality)
- ⚠️ **Screenplay Cards**: Buttons stack vertically (less convenient)
- ⚠️ **Scene Content**: Reduced scrolling height
- ⚠️ **General Layout**: More scrolling required

## Mobile Workflow

### 1. Prepare Your Device

#### Enable Auto-Rotation
- **Android**: Settings → Display → Auto-rotate screen
- **iOS**: Settings → Display & Brightness → View → Zoomed → Lock Rotation (off)

#### Use Desktop View
- **Chrome Mobile**: Menu (⋮) → Desktop site
- **Safari Mobile**: Share icon → Request Desktop Site

### 2. Optimize Your Setup

#### Recommended Device Types
- **Tablets** (iPad, Android tablets) - Best experience
- **Large Phones** (6"+ screens) - Good experience
- **Small Phones** (5"+ screens) - Usable but limited

#### Screen Size Recommendations
- **Minimum**: 5" screen with horizontal orientation
- **Recommended**: 7"+ tablet or large phone
- **Optimal**: 10"+ tablet for extended writing

### 3. Writing Best Practices

#### Touch Typing
- Use external keyboard for longer sessions
- Consider Bluetooth keyboards for better typing experience
- Use voice typing when available

#### Navigation
- Use swipe gestures for scrolling
- Pinch to zoom for text adjustment
- Use back button for navigation

## Performance Tips

### Optimize Battery Life

- **Reduce brightness** - Lower screen brightness for longer battery
- **Close unused tabs** - Keep only essential tabs open
- **Use Wi-Fi** - Prefer Wi-Fi over cellular for better performance

- **Local storage** - Screen Dreams saves locally on device

## Troubleshooting

### Common Mobile Issues

#### "Interface is too small"
**Solution:** Use horizontal orientation and desktop view

#### "Typing is difficult"
**Solution:** Use external keyboard or voice typing

#### "Buttons are hard to tap"
**Solution:** Zoom in or use larger device

#### "Auto-save not working"
**Solution:** Check network connection and browser permissions

### Browser-Specific Issues

#### Safari iOS
- **Issue**: Some features may have limited functionality
- **Solution**: Use Chrome or request desktop site

#### Chrome Android
- **Issue**: Text selection can be tricky
- **Solution**: Use long press for text selection

#### Firefox Mobile
- **Issue**: Some gestures may not work
- **Solution**: Use touch-friendly alternatives
