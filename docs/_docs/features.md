# Screen Dreams Features

Explore the comprehensive feature set that makes Screen Dreams the perfect tool for screenwriters.

## Screenplay Editor

Professional screenplay editor with Fountain format support:

### Fountain Format Support
Write screenplays using the industry-standard Fountain markup language. Enjoy syntax highlighting, auto-formatting, and real-time preview.

```fountain
FADE IN:

INT. COFFEE SHOP - DAY

A cozy, warm coffee shop filled with the aroma of fresh coffee.

SARAH (30s), a writer with a laptop and a half-empty coffee cup, stares at her screen.

<center>SARAH</center>
    (frustrated)
    Why can't I think of anything good?

She closes her laptop dramatically.
```

### Auto-Save
Never lose your work with automatic saving every 15 seconds (configurable). Focus on writing while we handle the saving.

### Keyboard Shortcuts
Work faster with comprehensive keyboard shortcuts for formatting, navigation, and common actions.
