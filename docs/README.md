# Screen Dreams Documentation

This directory contains the GitHub Pages documentation for Screen Dreams.

## Structure

```
docs/
├── index.html          # Main documentation page
├── css/
│   └── docs.css         # Custom styles
├── js/
│   └── docs.js          # Interactive features
├── images/
│   ├── editor-interface.png
│   ├── character-management.png
│   ├── ai-assistant.png
│   ├── mobile-view.png
│   └── mobile-showcase.png
└── pages/
    ├── getting-started.html
    ├── features.html
    ├── ai-assistant.html
    └── mobile-guide.html
```

## Setup

1. Enable GitHub Pages in your repository settings
2. Choose "Deploy from a branch"
3. Select `main` branch and `/docs` folder
4. Your documentation will be available at:
   `https://opensourcemechanic.github.io/screen-dreams`

## Adding Screenshots

1. Add screenshots to `docs/images/`
2. Use appropriate file names
3. Optimize images for web (max width: 1200px)
4. Use WebP format for better compression

## Customization

- Edit `css/docs.css` for styling changes
- Edit `index.html` for content updates
- Add new pages in the `pages/` directory

## Deployment

Changes to the `docs/` folder are automatically deployed when pushed to the main branch.
